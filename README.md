# KubeIaaS

KubeIaaS是基于Kubernetes的轻量化虚拟机管理套件，旨在充分利用Kubernetes的能力针对虚拟机应用场景进行扩展，以实现调度能力一体化、运维能力一体化、管控能力组件化、组件部署容器化。
